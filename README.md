# Minesweeper

Классическая игра «Сапёр», написанная на **C++** с использованием фреймворка **Qt**.

![Minesweeper Preview]()  

---

## Особенности

- **Настраиваемое поле**: при запуске можно настроить размер поля и количество мин
- **Безопасный первый клик**: мины генерируются только после первого хода, исключая стартовую клетку и её соседей
- **Алгоритм заливки**: автоматическое открытие пустых областей, с помощью алгоритма DFS
- **Флажки**: правая кнопка мыши для отметки мин

---

## Используемые технологии

| Компонент | Описание |
|-----------|----------|
| **Язык** | C++17 |
| **Фреймворк** | Qt 5/6 |
| **Сборка** | qmake / Qt Creator |
| **Алгоритмы** | DFS |
| **Структуры данных** | `std::vector`, `std::set`, `std::pair` |

---
### Требования для запуска
- Qt Creator 5.0 или выше
- Компилятор: MinGW / GCC / Clang
